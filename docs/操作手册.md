## 1. 系统概述

### 1.1 系统简介

在线教育平台是一套基于 Qt 6 框架开发的现代化桌面教学系统，提供完整的在线教学闭环：

- **教学端**：教师可创建课程、直播授课、布置作业、批改评分
- **学习端**：学生可浏览课程、观看直播/回放、参与互动、提价作业

### 1.2 核心功能一览

| 功能模块 | 教师 | 学生 |
|---------|:----:|:----:|
| 课程管理 | ✅ 创建/编辑/发布 | ✅ 浏览/搜索 |
| 直播课堂 | ✅ 推流授课 | ✅ 观看+互动 |
| 录播回放 | ✅ 管理课时 | ✅ 倍速回看 |
| 互动白板 | ✅ 板书教学 | ✅ 查看板书 |
| 弹幕互动 | ✅ 查看弹幕 | ✅ 发送弹幕 |
| 作业系统 | ✅ 布置/批改 | ✅ 提价/查分 |

### 1.3 运行环境

- **操作系统**：Windows 10+ / macOS 12+ / Linux (Ubuntu 22.04+)
- **运行时**：需安装 Qt 6.x 运行时库

## 3. 教师功能详解

### 3.1 课程管理

#### 创建新课程

1. 在仪表盘点击「课程列表」卡片
2. 点击右上角「+ 新建课程」按钮
3. 在弹出的对话框中填写：
   - **课程标题**（必填）
   - **课程描述**（可选）
   - **课程分类**（如：编程、设计、数学）
4. 点击「OK」确认创建

#### 管理课程

在课程列表页，您可以：

- **查看课程**：点击任意课程卡片进入详情页
- **发布课程**：课程默认创建为「草稿」状态
- **删除课程**：当前版本请在数据库中操作

#### 添加课时

1. 进入课程详情页
2. 点击右上角「+ 添加课时」按钮
3. 填写：
   - **课时标题**
   - **类型**：选择「直播(live)」或「录播(replay)」
   - **排序序号**
4. 点击「OK」确认

## 4. 学生功能详解

### 4.1 浏览课程

1. 从仪表盘点击「课程列表」卡片
2. 查看所有已发布的课程
3. 使用搜索框快速查找感兴趣课程
4. 点击课程卡片进入详情页

### 4.2 观看课程

1. 在课程详情页查看所有课时
2. 点击「观看回放」按钮进入播放页
3. 使用播放器控制条：
   - ▶️/⏸️ 播放/暂停
   - ⏪ 快退 / ⏩ 快进
   - 进度条拖拽
   - 倍速选择 (0.5x / 1.0x / 1.5x / 2.0x)
   - 音量调节
4. 系统自动记录学习进度

### 4.3 参与直播

详见 [第5章：直播教室指南](#5-直播教室指南)

### 5.3 直播控制

- **播放/停止**：点击底部的「▶ 播放」或「■ 停止」按钮
- **音量调节**：拖动音量滑块

- **返回**：点击顶部「← 返回」按钮或按 `Esc` 键

## 7. 弹幕系统使用

### 7.1 发送弹幕

1. 在视频直播或回放页面底部找到弹幕输入框
2. 输入弹幕内容
3. 按 `Enter` 键或点击「发送」按钮
4. 弹幕将从右向左滚动显示

### 7.2 弹幕类型

| 类型 | 效果 | 默认颜色 |
|------|------|---------|
| **滚动弹幕** | 从右向左滚动 | #FFFFFF 白色 |
| **顶部固定** | 悬停在画面上方（淡入淡出） | - |
| **底部固定** | 悬停在画面下方（淡入淡出） | - |

### 7.3 弹幕控制

- **开关**：点击弹幕输入框旁的 📺/🚫 按钮切换显示
- **颜色**：当前版本使用默认白色（后续版本可自定义）
- **字号**：默认 24px（后续版本可调节）

## 8. 作业系统指南

### 8.1 教师：布置作业

1. 进入课程详情页
2. 目前需通过代码调用布置（后续版本增加 UI 界面）
3. 批改作业：在「作业管理」页面点击「批改」按钮

### 8.2 教师：批改作业

1. 从仪表盘进入「在线作业」
2. 点击作业旁的「批改」按钮
3. 在学生提价列表中点击对应学生的「批改」按钮
4. 填写：
   - **评分**（0-100 分）
   - **评语**（文字反馈）
5. 点击「OK」确认提交

### 8.3 学生：提价作业

1. 从仪表盘进入「在线作业」
2. 点击作业旁的「提价作业」按钮
3. 在弹出的对话框中：
   - 输入**作业内容**（文本）
   - 可选上传**附件**（PDF/图片/ZIP）
4. 点击「提价作业」按钮提交

### 8.4 状态说明

| 状态 | 含义 | 操作 |
|------|------|------|
| 未提价 | 作业已布置，学生尚未提交 | 学生可提价 |
| 已提价 | 学生已提交作业 | 等待教师批改 |
| 已批改 | 教师已完成评分 | 查看成绩与评语 |

## 9. 常见问题

### Q1: 启动后提示缺少 Qt Multimedia 模块？

请确保已安装完整的 Qt 6 运行时，包括 `qt6-multimedia` 和 `qt6-multimedia-widgets` 模块。

### Q2: 直播功能无法使用？

直播功能基于 `QMediaPlayer` 实现，支持的协议取决于系统解码器：
- 推荐使用 RTMP 流地址
- 本地测试可使用文件路径或 UDP 流
- 确保防火墙未阻止网络连接

### Q3: 数据库文件在哪里？

默认数据库文件 `edu_platform.db` 创建在应用同级目录下。
如需自定义路径，修改 `config.json` 中的 `database/path` 配置项。

### Q4: 如何重置数据？

删除 `edu_platform.db` 文件后重新启动应用，系统会自动重建数据库并初始化种子数据。

### Q5: 支持哪些视频格式？

取决于系统安装的编解码器。Windows 上推荐使用 MP4 (H.264 + AAC) 格式以获得最佳兼容性。
